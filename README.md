## 🐣 Featured projects

### 🚀 [Modern Data Pipeline](https://github.com/sm7ss/modern-data-pipeline)
End-to-end ETL pipeline that processes 244M rows with **30% less time and 45% less memory**. Make automatic decisions based on available resources.

### 📊 [Auto Insight Dual Analysis Tool](https://github.com/sm7ss/auto-insight-dual-analysis-tool)
Dual tool: automated analysis + interactive dashboard with dynamic filters, for HTML reports. **[Live demo](https://prism-77em.onrender.com)**

### 🧠 [Memory-Aware Data Loader](https://github.com/sm7ss/memory-aware-data-loader)
System that prevents OOM errors by analyzing files before uploading them and selecting the optimal strategy (eager/lazy/streaming).

### 📑 [Adaptive Data Pipeline](https://github.com/sm7ss/adaptive-data-pipeline)
Auto-scaling between Polars Eager, Lazy and Ray Distributed according to data size vs available RAM, does not break data pre-processing or cleaning processes.

### 🧹 [Data Cleaning Pipeline](https://github.com/sm7ss/data-cleaning-pipeline)
Automated cleaning with Prefect orchestration, hierarchical null handling and Pydantic validation.
